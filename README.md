## Overview

**International Payment System** is a full-stack international payments system for a bank created on visual studio code.  

**Backend** – ASP.NET Core Web API 
   - Handles customer and employee registration, login, and payment processing.  
   - Stores sensitive data securely using hashing, salting, and input validation.  
   - Connects to a SQL Server database via Entity Framework Core.  
   - Provides endpoints for customer payments, employee verification, and SWIFT submissions.

**Frontend** – React App 
   - Modern, responsive UI for customers and employees.  
   - Login/Register forms, payment forms, and dashboards.  
   - Connects to the backend API to display and submit transactions.  
   - Built with React Router and Tailwind CSS for a sleek design.
